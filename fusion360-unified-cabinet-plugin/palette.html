Thickness → Board Type / 按厚度归类

            

              
Configure nominal thickness for each board type. Save Rules stores on this design; Set as Default stores a user-wide default for designs that have no saved rules. Apply writes matching boardType / materialClass into Fusion.

              

              

                
                
              

              
              
              
unknown / empty does not count as existing — those are always filled. Overwrite only replaces a known board type (built-in or custom).

              

                
                
                
              

              
Default: carcass 15 · partition 18 · door 16 (±0.6 mm).

3. Manual Override — Color / 手动颜色

            

              
Type a color or choose a saved user preset. Apply creates a manual Color lock. Saving a preset only updates this machine's color library and does not modify Fusion panels.

              
              
              
Select doors (or their parent assembly), enter a color name, then apply.

              

                
                
                
                
              

            

          


          

            
2. Auto Analyze — Hinge Propagation

            

              
Finds blind circular hinge cups geometrically. The face they open onto is the back = MILLING. That back is copied to other selected panels that are coplanar and same-orientation. Opposite broad face becomes NON_MILLING.

              
Select panels (or their parent assembly), then propagate.

              

                
              

            

          


          

            
2. Auto Analyze — Milling Surface / 铣削面

            

              

                

                  
Writes immediately to unlocked panels. Geometry analysis for every selected body: hinge cups, then half-slots; no evidence → EITHER only when unassigned. Manual face-up locks are skipped. Select Milling Faces is read-only.

                  
Select bodies (or their parent assembly), then analyze.

                  

                    
                    
                  

                

                

                  

                    Diagnostics / 诊断区
                    
                  

                  
Run Analyze Selection to see per-body results (why skipped, decision source).

                

              

            

          


          

            
2. Auto Analyze — Door Orientation / 门板朝向

            

              
Orient writes unlocked doors immediately. Hinge/slot evidence wins, then assembly/view votes. Select Colour Faces is read-only. Revert Panel Surface is a manual override and locks that body's face-up until Reset to Auto.

              

                
                
              

              

                
                
              

              
Manual point: not set.

              
Select furniture (assemblies and loose panels), then orient.

              

                
                
                
                
              

            

          


          

            
4. Inspect — Hinge Diagnostics / 只读诊断

            

              
Read-only check for debugging wrong hinge/colour faces. For each selected panel it reports: which broad face the hinge cups open onto (A/B), how that was decided (topology / inner loop / floor normal), the MILLING role currently stored on each face, and whether they agree. Copy the report and paste it back for troubleshooting. 只读诊断，不会修改任何数据。

              
Select panels (or their parent assembly), then diagnose.
